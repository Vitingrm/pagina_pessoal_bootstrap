# Página Pessoal - Victor Gabriel Resende Martins

Este projeto contém uma página pessoal desenvolvida utilizando HTML, CSS e JavaScript, com duas versões distintas utilizando diferentes frameworks CSS: Bootstrap e Tailwind CSS.

## Estrutura do Projeto

- `bootstrap-version/`: Versão utilizando Bootstrap
  - `index.html`: Página principal
  - `css/styles.css`: Estilos customizados para temas
  - `js/script.js`: JavaScript para alternância de tema

## Funcionalidades

- Layout responsivo
- Tema claro e escuro com botão de alternância
- Informações pessoais: nome, formação, interesses, projetos e contatos

## Como Visualizar

Abra os arquivos `index.html` em um navegador web.

Link do Github Pages: https://vitingrm.github.io/pagina_pessoal_bootstrap/

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Tailwind CSS
